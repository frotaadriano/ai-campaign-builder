# Master Plan

## Fases
- Phase 1 ? Canvas foundation
- Phase 2 ? Story model and persistence
- Phase 3 ? AI mock integration
- Phase 4 ? Real AI integration
- Phase 5 ? UX polish and extensibility

## Objetivo de cada fase
- Phase 1: construir o canvas funcional sem AI
- Phase 2: implementar modelos e persistencia local
- Phase 3: integrar geracao mock e regeneracao parcial
- Phase 4: integrar AI real com controle de contexto
- Phase 5: polir UX e preparar extensibilidade

## O que e MVP
- Canvas com blocos conectados
- Persistencia local de campanhas
- Regeneracao parcial (mock ou real)
- Fluxo completo sem travar

## Dependencias entre fases
- Phase 2 depende de Phase 1
- Phase 3 depende de Phase 2
- Phase 4 depende de Phase 3
- Phase 5 depende de Phase 4
